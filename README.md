# Price-Forecast Dashboard (Gas • Housing • Groceries)

Interactive Plotly dashboards that visualize historical trends (2020 → today) and 24-month forecasts (to 2027) for three everyday cost buckets in the United States:

1. 🏠 Housing Market Index (VNQ proxy)  
2. ⛽ Gas Price Index (USO proxy)  
3. 🍎 Grocery Price Index (DBA proxy)

The model enriches each series with key macro-economic drivers (10-year Treasury yield, WTI crude, CPI food-at-home) and simple seasonality features, then predicts the next two years with a tuned Random Forest.
